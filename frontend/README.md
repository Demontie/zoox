## Especificações
- Node >= 12.18.3
- Docker >= 19.03.12
- yarn >= 1.22.5

## Instalação na maquina local

```bash
# Clonar repositório
$ git clone https://github.com/Demontie/zoox.git

# Pasta do frontend
$ cd frontend

# Variáveis de ambiente
$ cp .env.example .env

# Instalar dependências
$ yarn install

# Execute a aplicação
$ yarn serve
```
